What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages.
It structures web content by organizing text, images, videos, links, and other elements.

Key Features of HTML:

Defines the structure of a web page
Organizes content such as text, images, and videos
Creates links and navigation
Works with CSS and JavaScript to build interactive websites

Basic HTML Elements:

html – The root element of an HTML page
head – Contains metadata and styles
body – Holds the visible content of the webpage
h1 - h6 – Heading elements
p – Paragraph element
a – Anchor tag for links
img – Adds images
ul, ol, li – Creates lists

How Does HTML Work?

A web browser reads HTML code and displays the corresponding elements on the screen.
Combined with CSS (Cascading Style Sheets) and JavaScript,
HTML helps build visually appealing and dynamic websites.
